# El jardin de las delicias - Explorador

Explorador visual estatico de detalles historicos de la pintura *El jardin de las delicias*.

## Tecnologia

- HTML
- CSS
- JavaScript

No usa React, Next.js, Node, Vite, Vinext, base de datos ni proceso de build.

## Archivos principales

- `index.html`: estructura del sitio
- `styles.css`: estilos visuales y responsive
- `script.js`: interaccion, filtros, lente y fichas de detalle
- `public/favicon.svg`: icono del sitio

## Uso local

Abrir `index.html` en el navegador.
